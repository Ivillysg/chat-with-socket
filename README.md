## Instruções

```bash
# Clonar o repositório
git clone https://github.com/Ivillysg/chat-with-socket.git

# Primeiro iniciar o servidor
cd server

# Baixar as dependências
yarn ou npm i

# Para executar
yarn dev
ou
npm run dev

# Depois iniciar o cliente
cd client

# Baixar as dependências
yarn ou npm i

# Para executar
yarn start
ou
npm run start

```
